# MedicalClinicPOO
Medical Clinic Management - Java OOP Project

This project is a simple Java OOP application to manage a medical clinic.

Features:
- Patient management
- Doctor management
- Appointment scheduling
- Medical record tracking
- Prescription storage

OOP Concepts Used:
- Encapsulation
- Classes and Objects
- Constructors
- Object relationships
- ArrayList collections

Run:
Compile all files and run Main.java
